# Chaos im LABOR mit LuXeria Website

## Wichtige Seiten:

- Konfiguration: `_config.yaml`
- Titelseite: `index.html`
- Navigation: `_data/navigation.yml`
- Seiten: `_pages/`

## Build

```console
$ bundle config set --local path 'vendor/bundle'
$ bundle install
$ bundle exex jekyll serve
```
